Welcome to ARLE's documentation!
================================

ARLE or Automated Robot for Library Enhancement is an effort to make an automated robot capable of mananging (picking and placing) books in the institute library. It is a complex project making use of various technologies like - Computer Vision, Deep Learning, Autonomous Navigation, Localization, 3D Mapping, Sensor Data Management, Robot Operating System and many more. The project will be completed in stages.

At present, the project is in its first stage, which aims to make an four wheeled platform vehicle capable of autonomous navigation.
